## 集成到主入口

### 使用示例

```typescript
import { LobeHubOpenClaw } from './src';

const system = await LobeHubOpenClaw.create({
  openclaw: {
    gatewayUrl: 'http://localhost:18789',
    workspaceDir: './workspace',
  },
  lobehub: {
    baseUrl: 'http://localhost:3210',
  },
  integrations: {
    sharedMemory: true,
    sharedSkills: true,
    crossTrigger: true,
  },
});

// 检查状态
const status = await system.status();
console.log(status);
```
